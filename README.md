# Login Page 2

Alternative LexReason login page with a brighter editorial legal-office style.

## Files

- `index.html` - deployable static login page
- `reference-legal-desk.jpg` - reference image used for the hero panel
- `.github/workflows/deploy.yml` - GitHub Pages deployment workflow

## Deploy

1. Push this folder contents to the GitHub repository.
2. In GitHub, open `Settings > Pages`.
3. Set the source to `GitHub Actions`.
4. The workflow will publish the site on pushes to `main`.
